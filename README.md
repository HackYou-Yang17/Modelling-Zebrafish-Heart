# How To Run The Project
1.  If the environment is not activated yet (the left side of the terminal says [base] and not [modelling]) then input:

    conda activate modelling

2. To access the correct folder, input:

    cd src

3. To run the model, input:

    python simulate.py

4. To visualise the model, input:

    python visualise.py


# How To View The Markdown Files In 'Knowledge'
1.  Make sure Markdown Preview Enhanced is downloaded as an extension.

2.  To access the preview, press:

    ctrl + shift + v
